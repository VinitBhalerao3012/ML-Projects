# ML-Projects
Collection of machine learning projects including weather prediction, stock price prediction, cancer diagnosis, wine quality analysis, and marketing data analysis.

# Machine Learning Projects Portfolio

This repository contains a collection of real-world Machine Learning projects implemented using Python.  
Each project focuses on data analysis, visualization, feature engineering, and predictive modeling.

## Projects Overview

### 1️. Weather Prediction
**Folder:** `Weather Prediction`

- Dataset: Seattle Weather Dataset
- Tasks:
  - Exploratory Data Analysis (EDA)
  - Feature preprocessing
  - Weather condition prediction
- Techniques:
  - Pandas, NumPy
  - Data visualization using Matplotlib & Seaborn
  - Machine Learning models

### 2. Wine Quality Prediction
**Folder:** `Wine Quality Prediction`

- Tasks:
  - Data cleaning and preprocessing

  - Wine quality classification

  - Random Forest

- Visualizations:
  - Bar plots
